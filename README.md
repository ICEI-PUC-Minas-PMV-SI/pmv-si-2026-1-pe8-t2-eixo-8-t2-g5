# Sistema de Controle de Estoque para Autopeças

**CURSO:** Sistemas de Informação  
**DISCIPLINA:** Projeto - Trabalho de Conclusão de Curso  
**SEMESTRE:** 8º  

O presente projeto tem como objetivo o desenvolvimento de um sistema de informação voltado para o controle de estoque de uma empresa do segmento de autopeças localizada em Belo Horizonte. A proposta surgiu a partir da identificação de dificuldades relacionadas à organização das informações de estoque, controle de movimentações e acompanhamento das vendas realizadas pela empresa.

O sistema será desenvolvido com foco na centralização das informações, permitindo maior controle sobre produtos, entradas, saídas e movimentações de estoque. Além disso, a solução busca auxiliar na tomada de decisão da empresa, reduzindo falhas operacionais e melhorando a organização dos processos relacionados ao atendimento e reposição de produtos.
